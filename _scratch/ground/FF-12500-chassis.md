FAMILY: HPE FlexFabric 12500 Switches (E3 "HPE FlexFabric 12500 Switches")
- OEM source: HPE-authored data sheet **4AA3-0666ENW, February 2016, Rev. 15** ("HPE 12500 Switch Series"), full HPE-hosted PDF recovered (Akamai/psnow c04111591 = RETIRED/obsolete). Cross-checked vs H3C S12500 OEM twin (h3c.com installation guide + S12508-AC manual) — slot architecture matches.
- Operating temperature (all PIDs, verbatim): **32°F to 104°F (0°C to 40°C)**; storage −40°C to 70°C; humidity 5%–95% noncondensing.
- EOL: HPE psnow QuickSpec c04111591 officially **RETIRED — "base products… now considered obsolete and… no longer available for sale"** (last datasheet rev Feb-2016). Exact EOS/EOSL dates are NOT in any OEM doc reached → EOL date ZU_VERIFIZIEREN (OEM date not published in reachable docs).
- Comware / branding (verbatim): **Comware v7** ("modern Comware v7 Operating System"); MPU JG497A = "Type A Main Processing Unit with Comware v7"; architecture non-blocking distributed Clos, IRF, MDC, EVI, OpenFlow 1.3. Layer = **L2 + L3** (full IPv4/IPv6 routing, MPLS/VPLS) for all PIDs.
- ⚠️ ROSTER FINDING: the datasheet documents TWO generations. The Phase-1 six are original-gen. The datasheet's spec tables ALSO document a **12500E generation absent from Phase-1**: JG782A (12508E AC), JG783A (12508E DC), JG784A (12518E AC), JG785A (12518E DC) — the E-gen is what delivers the family headline 10.8 Tb/s (12508E) / 24.3 Tb/s + 10.8 Bpps (12518E). FLAG for operator: decide whether the 12500E chassis belong in this family denominator (they share E3 "12500" but are a distinct HW gen with their own JG-fabric modules JG798A/JG800A + JG802A MPU).
- ⚠️ Redundancy (verbatim, all): "full hardware redundancy for each component including power supplies, fan trays, supervisor modules, and fabric modules." PSU redundancy N+1/N+N per config; 2 MPU slots = 1+1 supervisor redundancy; fabric N+1.
- ⚠️ SwK NOTE: datasheet lists "Routing/Switching capacity" per model (below) — verbatim, never port-summed. Family max "24.3 Tb/s… 10.8 Bpps… up to 400 Gb/s per line card slot" applies to the top E-gen chassis (12518E), not to every PID.
- MPU/Supervisor PIDs (series-level, verbatim): **JC072B** "HPE 12500 Main Processing Unit"; **JG497A** "HPE 12500 Type A Main Processing Unit with Comware v7 Operating System"; **JG802A** "HPE FlexFabric 12500E Main Processing Unit" (E-gen). (2 MPU slots per chassis; datasheet does not per-chassis-restrict MPU PID beyond gen — E-gen chassis use JG802A.)
- Weight note: all chassis >50 kg (verbatim), so no ZU_VERIFIZIEREN needed.
