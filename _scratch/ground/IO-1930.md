## FAMILY BLOCK
- **Operating temperature:** 32°F to 104°F (0°C to 40°C) — all models. Non-operating/storage: -40°F to 158°F (-40°C to 70°C). Op. humidity 15% to 95% @ 40°C. Altitude up to 10,000 ft (3 km).
- **IRF/stacking:** none — the 1930 series does NOT stack (no stacking feature in datasheet; stacking is a 1960-only capability). Link aggregation only (LACP, up to 8 ports/trunk; 8-port=4 trunks, 24-port=8 trunks, 48-port=16 trunks).

- **Layer class:** Smart-managed **Layer 2+** (Layer 2 + Layer 3 IPv4 static routing, up to 32 static routes; IPv6 host mode; ACLs; up to 256 VLANs, ID range 2–4093).
- **PoE standard:** Ethernet Alliance Certified Class 4 PoE (PoE+; **IEEE 802.3at**), up to 30 W/port; also powers 15.4 W 802.3af devices. (Not 802.3bt.)
- **OS/management:** Aruba Instant On cloud management (mobile app + cloud-based web portal); local Web GUI; SNMP v1/v2c/v3. Device Management string (verbatim): "Aruba Instant On Portal; Web browser; SNMP Manager". Cloud-managed — no on-box enterprise CLI. TPM-based security.
- **EOL:** Current/shipping family (no EoS notice published as of this grounding). Warranty = Aruba Instant On Limited Lifetime (24×7 phone support first 90 days, chat support for warranty life) — NOT enterprise Foundation Care. Positioning flag: low-end SMB, cloud-portal-managed, not "neu-versiegelt" enterprise.
- **Form factor:** all 1U desktop/rack-mountable. Rack kit **HPE X410 1U Universal 4-post Rack Mounting Kit (J9583A)** listed as accessory for 24-/48-port models (JL682A–JL686A); 8-port models (JL680A/681A) desktop/wall (no rack kit listed).
- **Roster VERIFIED:** exactly the 7 base PIDs JL680A–JL686A (matches Phase-1 manifest; no addition/removal). B-revs JL683B / JL684B / JL686B = HW-refresh, SAME spec — carry as A/B same-spec variants (confirmed by vendor listings under identical datasheet specs; datasheet itself lists only A-revs).
